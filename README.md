# Projektplan

## 1. Projektbeskrivning
Projektet ska vara ett forum, tänkt vara likt flashback eller reddit. Forumet ska ge användare funktionen att registrera sig, logga in, 
skapa inlägg, kommentera på andra inlägg samt kunna redigera sitt konto såsom namn, email.
## 2. Vyer (sidor)
Appen ska ha en hemsida, där man kunna se alla inlägg som blivit upplagda samt att kunna logga in eller registrera sig på den sidan.
Den ska även ha sidor
